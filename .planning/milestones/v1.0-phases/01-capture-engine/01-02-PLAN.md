Create `src/main/capture.ts` — the core capture engine with macOS permission gating and PNG output.

**src/main/capture.ts must contain:**

1. **`checkMacOSPermission()` function** (exported):
   - Returns `Promise<'granted' | 'denied' | 'unavailable'>`
   - On non-darwin platforms: return `'granted'` immediately
   - On darwin: call `systemPreferences.getMediaAccessStatus('screen')`
   - If `'granted'`: return `'granted'`
   - If `'not-determined'`: call `systemPreferences.askForMediaAccess('screen')`, return result
   - If `'denied'`: call `shell.openExternal('x-apple.systempreferences:com.apple.preference.security?Privacy_ScreenCapture')`, return `'denied'`
   - IMPORTANT: Must call getMediaAccessStatus on EVERY launch (not cached) — Sequoia re-prompts monthly (PLAT-05)

2. **`getScreenSources()` function** (exported):
   - Calls `desktopCapturer.getSources({ types: ['screen'], thumbnailSize: { width, height } })` where width/height come from the active display's actual pixel dimensions (`screen.getDisplayNearestPoint(screen.getCursorScreenPoint()).size`)
   - CRITICAL: Use `types: ['screen']` NOT `types: ['window']` — window type produces black screenshots on Chromium windows
   - Returns array of `{ id: string; thumbnail: string }` where thumbnail is `source.thumbnail.toDataURL()`
   - Wraps in try/catch for Wayland portal crash (PLAT-06): catch returns empty array and logs to stderr

3. **`captureRegion(rect: RegionRect)` function** (exported):
   - Gets the active display via `screen.getDisplayNearestPoint(screen.getCursorScreenPoint())`
   - Gets `scaleFactor` from that display
   - Multiplies rect x/y/width/height by scaleFactor to convert CSS pixels to physical pixels: `Math.round(rect.x * scaleFactor)` etc.
   - Calls `getScreenSources()` to get the screen thumbnail
   - Creates `nativeImage` from source, calls `.crop(physicalRect)`
   - Calls `.toPNG()` on the cropped image to get a Buffer
   - Creates temp dir: `path.join(os.tmpdir(), 'snapview')` with `fs.mkdir(tempDir, { recursive: true })`
   - Generates filename: `snapview-${Date.now()}-${crypto.randomBytes(8).toString('hex')}.png`
   - Writes PNG buffer to `path.join(tempDir, filename)` via `fs.writeFile`
   - Returns `{ filePath }` matching the `CaptureResult` interface

Import types from `../shared/types` using the `RegionRect` and `CaptureResult` interfaces.

**src/main/cleanup.ts must contain:**

1. **`sweepOldCaptures()` function** (exported):
   - `const TWENTY_FOUR_HOURS_MS = 24 * 60 * 60 * 1000`
   - Reads directory `path.join(os.tmpdir(), 'snapview')`
   - Filters entries: only files matching `snapview-*.png` pattern
   - For each matching file: `fs.stat()` to check `mtimeMs`
   - If `Date.now() - stat.mtimeMs > TWENTY_FOUR_HOURS_MS`: delete with `fs.unlink(fullPath).catch(() => {})` (best-effort)
   - Entire function wrapped in try/catch — if snapview dir doesn't exist, silently return
   - Must NOT block the critical path — caller should fire-and-forget (no await on critical path)

**src/main/index.ts must contain (in this order):**

1. **Linux GPU flags** (BEFORE app.whenReady — PLAT-02):

```typescript
if (process.platform === 'linux') {
  app.commandLine.appendSwitch('enable-transparent-visuals');
  app.commandLine.appendSwitch('disable-gpu');

```

2. **Wayland uncaughtException handler** (PLAT-06):
```typescript
process.on('uncaughtException', (err) => {
  console.error('[snapview] Uncaught exception:', err.message);
  app.exit(1);
});
```

3. **Hard-exit timeout** (PLAT-03) — set BEFORE any UI:
```typescript
const HARD_EXIT_TIMEOUT_MS = 30_000;
const hardExitTimer = setTimeout(() => {
  console.error('[snapview] Hard exit timeout reached — forcing quit');
  app.exit(1);
}, HARD_EXIT_TIMEOUT_MS);
hardExitTimer.unref();
```

4. **`createOverlay()` function** — creates BrowserWindow on active monitor (PLAT-04):
   - Call `screen.getCursorScreenPoint()` to get cursor position
   - Call `screen.getDisplayNearestPoint(cursorPos)` to get the active display
   - Extract `{ x, y, width, height }` from `activeDisplay.bounds`
   - Create BrowserWindow with these exact options:
     ```typescript
     {
       x, y, width, height,
       frame: false,
       transparent: true,
       alwaysOnTop: true,
       fullscreen: false,  // CRITICAL: fullscreen:true ignores x/y and uses primary display
       skipTaskbar: true,
       webPreferences: {
         preload: path.join(__dirname, '../preload/preload.js'),
         contextIsolation: true,
         nodeIntegration: false,
         sandbox: false,  // Required for preload IPC
       },
     }
     ```
   - Load renderer: `overlay.loadFile(path.join(__dirname, '../renderer/index.html'))`
   - Return the BrowserWindow

5. **IPC handlers** — registered with `ipcMain.handle` using channel names from `IPC_CHANNELS`:
   - `capture:get-sources`: calls `getScreenSources()` from capture module, returns sources array. If `checkMacOSPermission()` returns `'denied'`, return `{ permissionDenied: true }` instead.
   - `capture:region`: calls `captureRegion(rect)` from capture module, emits `result.filePath` to stdout via `process.stdout.write(result.filePath + '\n')`, clears hardExitTimer, calls `app.quit()` with exit code 0
   - `capture:cancel`: clears hardExitTimer, calls `app.quit()` with exit code 2 (no stdout output)

6. **App lifecycle:**
   - `app.whenReady()`: fire `sweepOldCaptures()` (no await — background), then call `createOverlay()`
   - `app.on('window-all-closed', () => app.quit())` — prevents orphaned process (PLAT-03)

7. **Exit code handling:**
   - Exit code 0: success (file path was emitted to stdout)
   - Exit code 1: error (permission denied, capture failed, hard timeout)
   - Exit code 2: user cancelled (ESC, window close without capture)
   - Use `app.on('before-quit', ...)` or `process.exitCode` to set the appropriate code

**src/preload/preload.ts must contain:**

The contextBridge that exposes exactly three IPC channels to the renderer:

```typescript
import { contextBridge, ipcRenderer } from 'electron';

contextBridge.exposeInMainWorld('snapviewBridge', {
  getSources: (): Promise<{ id: string; thumbnail: string }[] | { permissionDenied: true }> =>
    ipcRenderer.invoke('capture:get-sources'),
  captureRegion: (rect: { x: number; y: number; width: number; height: number }): Promise<{ filePath: string }> =>
    ipcRenderer.invoke('capture:region', rect),
  cancel: (): Promise<void> =>
    ipcRenderer.invoke('capture:cancel'),
});
```

Import channel names from shared types if the import resolves in preload context, otherwise use string literals matching `IPC_CHANNELS` values.

IMPORTANT: The renderer must NEVER import from 'electron' directly. All privileged operations go through this bridge.
